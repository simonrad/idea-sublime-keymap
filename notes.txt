Copy the "MyLayout.keylayout" file to "~/Library/Keyboard Layouts", then go to
System Preferences -> Keyboard -> Input Sources -> "+" icon -> Others -> add My Layout.
This keyboard layout prevents "option + i" from outputting a character.
